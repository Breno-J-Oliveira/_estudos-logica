## 📖 Sobre

Aula introdutória ao sistema de versionamento **Git**. Prática de comandos essenciais: `init`, `add`, `commit`, `branch`, `merge` e `push`.

## 💻 Comandos Praticados

```bash
git init                    # Inicializar repositório
git add .                   # Adicionar arquivos
git commit -m "mensagem"    # Criar commit
git branch feature-x        # Criar branch
git checkout feature-x      # Mudar de branch
git merge feature-x         # Mesclar branch
```
